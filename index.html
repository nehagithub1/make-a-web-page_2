Lesson 2: Creating a Structured Document with HTML

Structure of a Page

Computer code is structured in a way that allows communication with the browser telling it how to display the page. It classifies the various aspect of the page (text, image etc) and uses TAGS that convey not only the type of content but how the content will be displayed.

For example, an h1 tag tells the browser that the text needs to be displayed as a heading 1. And using the tree like structure, we can use the code to tell the browser to display the h1 tag under a specified div tag which will clarify where (which box) the h1 tag should be under.

Each element has an opening TAG and a closing TAG which allows the coder to specify where its attributes begin and end.

Avoiding Repetition – If repetition is not avoided, programmers have to rewrite and recheck the code every time when they want to make a change. Especially in CSS, as styling can be applied in various areas of the page and if a change needs to be made the coder can change the style itself and have it automatically be applied to ALL the components that need that particular style.
